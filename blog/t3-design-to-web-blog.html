EDA Week 2

Responsive

A responsive site changes its layout to suit the size of a device. All sites should be clear and readable on a mobile as well as a big desktop - which sometimes means certain elements need to be resized, rearranged or removed completely.

What mobile first design is, and why it's important.

I think good desiign these days starts with the mobile design first so the mobile design doesn't have to navigate lots of unneccesary CSS.

Frameworks

Frameworks are out-of-the-boxs tools that do a lot of the ground work for you to make a nice design. If you fit in with their rules you can create beautiful looking pages using frameworks. Unfortunately I don't like fitting in with assigned rules. The big con for me was not being able to edit the skeleton CSS (as we were linking to an online stylesheet)so I had to continuously find workarounds to hack the trickier things I wanted, which in the end won't make for good code. This system today of having 3 stylesheets on top of each other seems complicated and breakable.

Wireframes

Wireframes are very very simple layout-based pictures that help clients get an idea of what a designer is thinking; help designers figure out how their layout will work in code; and help developers code a design.
